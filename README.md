# pet-shop

## What´s the purpose of this application
Since the there is a vast black market for dog trading,
dog owners need a platform to track where their new pet
coming from and have been to. 
This applications goal is to easily adopt a dog with your 
meta mask account.

## What does this project do
You can adopt a dog and pay with your meta mask account.

## How does it work
Install Truffle if not already happened.  
Cd to the root.  
Truffle migrate.  
Truffle compile.  
Npm run dev.  
Follow the link.   
If you didn´t add the meta mask extension, do it now.
Allow your browser connecting your meta mask extension.
